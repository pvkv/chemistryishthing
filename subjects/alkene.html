Алкени

ненасичені вуглеводні

безоксигенно вмісні

Загальна формула:

CnH2n, n≥2.

Типи ізомерії:

структурна, міжкласова (ізомери циклоалканам) ізомерія.

Довжина зв'язків:

між подвійним зв'язком атомів Карбону - 0.134нм, одинарного - 0.154нм.

Хімічні зв'язки:

сігма та пі зв'язки.

Застосування:

використовуються у медицині, як розчинник.

Агрегатний стан:

перші 3 гомологи - гази, наступні 14 - рідини, далі - тверді речовини.

Тип гібридизації

sp2 гібридизація; валентний кут 120°.

  1. Для утворення систематичної назви ненасиченого вуглеводню нерозгалуженої будови:
    1. визначають атом Карбону з найменшим локантом, біля якого розташований кратний зв'язок;
    2. замінюють суфікс -ан алкану на суфікс -ен алкену.
  2. Для утворення систематичної назви ненасиченого вуглеводню розгалуженої будови:
    1. вибирають найдовший карбоновий ланцюг, який містить кратний зв'язок;
    2. нумерацію починають з того кінця, до якого найближче розташований кратний зв'язок (згідно з принципом найменших локантів);
      • якщо в молекулі сполуки кратний зв'язок рівновіддалений від обох кінців карбонового ланцюга, то нумерацію починають з того кінця, до якого ближче розташований алкільний замісник;
      • якщо в молекулі сполуки кратний зв'язок рівновіддалений від обох кінців карбонового ланцюга, то у виборі нумерації користуються алфавітним порядком (етил —> метил —> пропіл).
  3. У назві сполуки зазначають:
    1. замісники, перераховані в алфавітному порядку, як й у випадку з алканами (відокремлені один від одного комами без пробілів, а від кореня ненасиченого вуглеводню - дефісом);
    2. назву ненасиченого вуглеводню, яка відловідае головному карбоновому ланцюгу (корінь) + ближчий до початку локант кратного зв'язку + суфікс -ен, які відокремлюються один від одного дефісами:
  1. Галогенування: \(\ce{C_nH2_n + Hal2 -> C_nH2_nHal2}\)
  2. Гідрогенізація: \(\ce{C_nH2_n + H2 -> C_nH_{2n+2} }\)
  3. Гідроґеногалогенування: \(\ce{C_nH2_n + HHal2 -> C_nH_{2n+1}Hal}\)
  4. Гідратація: \(\ce{C_nH2_n + HOH -> C_nH_{2n+1}OH}\)
  5. Горіння: \(\ce{CH2\bond{2}CH2 + 3O2 -> 2CO2 + 2H2O}\)
  6. Часткове окиснення (якісна реакція): \(\ce{CH2\bond{2}CH2 ->[KMnO4] OH\bond{1}CH2\bond{1}CH2\bond{1}OH}\)
  7. Полімеризація: \(\ce{nCH2\bond{2}CH2 -> (\bond{1}CH2\bond{1}CH2\bond{1})_n}\)
  1. Крекінг алканів: \(\ce{C8H18 ->[t°] C4H10 + C4H8}\)
  2. Дегідрування алканів: \(\ce{CH3\bond{1}CH3 -> CH2\bond{2}CH2 +H2 ^}\)
  3. Відщеплення галогеноводнів при розчині лугів: \(\ce{CH3\bond{1}CH2\bond{1}CHBr\bond{1}CH3 + KOH -> CH3\bond{1}CH\bond{2}CH\bond{1}CH3 + KBr + H2)}\)
  4. Дегалогенування дигалогеноалканів: \(\ce{CH2Br\bond{1}CHBr\bond{1}CH3 + Mg -> CH2\bond{2}CH2\bond{1}CH3 + MgBr2}\)
  5. Дегідратація спиртів: \(\ce{C2H5OH ->[t°, H2SO4] C2H4 + H2O}\)
